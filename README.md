# Sobre o projeto
Aplicação Backend do projeto HelpDesk, do curso Angular + Spring Boot.

Java, Spring Boot, API Rest e MySQL, utilizando o Banco de dados H2. Implementação de CRUD para Técnicos e Clientes, utilizando o padrão DTO para testar a API no Postman. 

A aplicação conta com um serviço de agendamento de Chamados.

Tratamento de exceções padronizadas.

Autenticação JWT.

## Tecnologias Utilizadas
• Java

• Spring Boot

• Spring Security

• JPA

• Bean Validation

• Maven

• Postman
